# Meetups With Firebase

## Getting Started

```bash
    # clone the project
    git clone <GIT-REPO-URL>
    cd meetups
    npm start

    # You must have a firebase project created to connect the app with a firebase realtime database by:
    # goto firebase.google.com 
    # click on Get started
    # click on Add project
    # enter the project name "react-meetups-app"
    # then click continue and then Save
    # finally click continue and go to the project overview dashboard
    # click on "build" then click "Realtime Database"
    # then click on "Create Database" leave the default region and click Next
    # Finally make sure to choose "Start in Test Mode" option to be able to manipulate the database then click on "Enable"
```
